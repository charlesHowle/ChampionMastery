# ChampionMastery
This python program for the online course CS50P, it allows an input in a GUI, which is sent to an API to return data.
The game "League of Legend" has many characters called champions that you can play. Each time you win or lose you gain some amount of mastery points with that champion. So, to view which champions someone is best at, we can determine this by simply looking at which champion they have the most mastery points. This python program is exactly this, with an intuative GUI to help.

Simply input the name of the player and his account tag and a list will show up. Want more to appear? Change the (10) in line 57 to have as many champions as you wish.

## Settup 
! YOU NEED YOUR OWN RIOT DEVELOPPER API KEY TO MAKE THIS WORK !
Sign in at https://developer.riotgames.com to get your own API key.
This key will expire after 24h so you need to regenerate one everytime it expires.

You may need to change both the "nbBaseURL" and "PLATFORM_URL" depending of the region of the account you are searching for.
